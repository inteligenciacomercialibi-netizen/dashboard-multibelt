🔗 GOOGLE SHEETS:

📊 Como conectar ao Google Sheets

Passo 1 — Crie a planilha no Google Sheets
A planilha precisa ter a aba chamada Dados com as seguintes colunas na linha 1:
dupla | segmento | meta | real | dev | margem
Cada linha subsequente representa uma combinação de dupla + segmento. Exemplo:
Luiz x Daiane | PVC | 50000 | 43000 | 1200 | 18
Luiz x Daiane | PU | 30000 | 28000 | 0 | 22
Welliton x Adrielly | BORRACHA | 60000 | 55000 | 800 | 15
Passo 2 — Publicar a planilha na web
1. No Google Sheets: Arquivo → Compartilhar → Publicar na web
2. Escolha a aba Dados e formato CSV
3. Clique em Publicar e copie o link gerado
4. O link tem este formato:
https://docs.google.com/spreadsheets/d/SEU_ID_AQUI/pub?gid=0&single=true&output=csv
5. Copie apenas o ID (parte em verde acima) e cole no campo "Google Sheets" no topo do dashboard.

Passo 3 — Atualização automática
O dashboard recarrega os dados da planilha a cada 2 minutos automaticamente. Você também pode clicar em ⬇️ Carregar para atualizar na hora.

⚠️ Importante: Os dados do local (localStorage) e da planilha são independentes. Ao carregar do Sheets, os dados locais são substituídos. Use o botão 💾 Backup antes de carregar se quiser preservar os dados locais.
